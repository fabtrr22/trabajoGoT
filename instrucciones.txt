Instrucciones para probar el proyecto


- Descargar el proyecto y abrirlo con VSCode o cualquier IDE
- abrir archivo script_creacion_bd.txt, copiar el script y pegarlo en una ventana de MySQL Workbench, para crear la base de datos
- Configurar app.py con el user y password que tengas de la base de datos para conectar. El mio es user: root, password: admin
- en una terminal de comandos, levantar el server en Flask app.py con el comando: python app.py
- acceder al server en el navegador http://127.0.0.1:5000/
- navegar solo en index.html y temporadas.html, esas son las unicas paginas renderizadas por ahora, si vas a personajes o contacto, el sitio da un 404 Not Found
- si todo se realizó correctamente, ya se podrían agregar temporadas

Prerrequisitos:

tener instalados Flask y MySQL (yo tengo MySQL Workbench)

Para instalar Flask: abrir una terminal y escribir pip install Flask
